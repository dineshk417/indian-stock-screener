# NiftyEdge — AI-Powered Indian Stock Screener

> Real-time trade signals, sentiment analysis, and a trade journal built for the Indian retail investor. Covers Nifty 50 and Nifty 200 across swing and intraday timeframes.

**Live app**: [stockscreener4.streamlit.app](https://stockscreener4.streamlit.app)

---

## What It Does

NiftyEdge is a multi-page Streamlit application that:

- Scans Nifty 50 / Nifty 200 stocks against 10 technical and fundamental strategies
- Generates swing (2–5 day) and intraday (9:15–15:30 IST) trade signals with entry, stop-loss, and two targets
- Analyzes 30+ market news headlines daily using Claude AI and surfaces a sentiment score, sector outlook, and actionable themes
- Logs every signal to a database, tracks outcomes against live prices, and shows P&L against a ₹1L base portfolio
- Sends Telegram alerts for new signals and a morning market briefing
- Includes a Tip Analyzer that evaluates WhatsApp/Telegram stock tips for pump risk and credibility

## Pages

### Home

- Today's signals feed: horizontal-scrolling cards for all signals generated today
- Top 5 gainers and losers from Nifty 50
- Animated market status indicator (open / pre-market / closed) in IST

### 1 · Market Overview
- Sector performance heatmap across all NSE sector indices
- Global markets snapshot: S&P 500, Dow Jones, Nasdaq, Crude Oil, Gold, USD/INR
- Market breadth gauge: advancing vs declining stock count
- YTD performance chart for major indices
- Live price fragments that refresh in place without a full page reload

### 2 · News & Sentiment
Powered by **Claude Sonnet** (VADER as fallback when API key is absent):
- Sentiment score 1–10 with a verbal label (Strongly Bullish → Strongly Bearish)
- Per-sector verdict: IT, Banking, Pharma, FMCG, Auto, Metal, Energy
- Bullish and bearish stock mentions extracted from headlines
- Key market themes, catalysts, and downside risks
- Actionable trade implications for the day
- Sources: Economic Times, Moneycontrol, Livemint, Business Standard, NSE, Google News RSS

### 3 · Fundamental Screener
- Filters: PE ratio, PB ratio, ROE, net profit margin, debt/equity, dividend yield
- Preset screeners: Value Picks, Quality Compounders, Dividend Stars
- Universe: Nifty 50 or Nifty 200
- Composite score: blends valuation, profitability, growth, and financial health (0–1)
- Sortable results table with all metrics and sector filter

### 4 · Technical Screener
- Filters: RSI range, trend direction, above SMA 50 / SMA 200, MACD bullish, volume spike 1.5×+
- Preset quick-scans: Oversold, Breakout, Golden Cross, Momentum
- Inline candlestick + RSI/MACD chart for any selected stock
- Detects: Golden Cross (event-based, within 15 trading days), Supertrend direction, MACD crossover

### 5 · Swing Trades
Scans the selected universe with 6 strategies and logs qualifying signals to the trade journal.

| Strategy | Condition |
|---|---|
| Trend Pullback | Price above SMA50 > SMA200, pulled back to EMA21, RSI 35–65 |
| Volume Breakout | Above SMA200, RSI 50–75, volume ≥ 1.5× average |
| Oversold Reversal | RSI < 40 and fundamental composite score > 0.35 |
| Bullish Setup | Above SMA200, MACD bullish, RSI < 70 |
| Golden Cross | SMA50 crossed above SMA200 within the last 20 bars |
| Supertrend Reversal | Supertrend flipped bullish within 5 bars + MACD bullish |

- Entry at close or limit; SL at 1.5× ATR below entry; T1 at 1:1.5 R, T2 at 1:2.5 R
- Minimum R:R enforced at 2.0; minimum confidence 1–5 stars filterable in sidebar
- Sentiment score from the News page adjusts signal bias
- One-trade-per-ticker rule: will not open a second position while one is already open

### 6 · Intraday Ideas
Active only during NSE market hours (9:15–15:30 IST). Auto-refreshes every 5 minutes.

| Strategy | Trigger |
|---|---|
| Opening Range Breakout | Break of first 15-min high/low with 1.5× volume |
| VWAP Bounce | Price bounces off VWAP with RSI between 40–60 |
| EMA Crossover | EMA9 crosses above EMA21 on 5-min chart with volume + VWAP |
| Supertrend Signal | Supertrend flips bullish on 5-min chart while price is above VWAP |

- Scans the 30 most liquid Nifty 50 stocks by recent average volume
- Shows a countdown timer with next open time when market is closed

### 7 · Signal Log & Performance
The trade journal for all signals generated by the app.

**Live Positions tab**
- Current price vs entry with real-time P&L (shown at trigger price when a level is hit, not drifted market price)
- Status: ACTIVE, NEAR SL, T1 HIT, T2 HIT, STOPPED, STALE
- Progress bar from SL to T2 with color coding
- Auto-resolves outcomes every 5 minutes during market hours; force-resolves stale signals from prior days

**Performance tab**
- Win rate % with breakdown: `12W (8T+4SQ) · 9L (7Stop+2SQ)`
- Net P&L after brokerage, STT, GST, exchange fees, stamp duty
- Expectancy (average R-multiple) with contextual label
- Portfolio card: current value vs ₹1,00,000 base capital with return %
- Outcome distribution pie chart and cumulative net P&L line chart
- Strategy breakdown table: wins, losses, open, win rate, avg R, net P&L

**History tab**
- Full table of all signals with entry, SL, T1, T2, exit price, outcome, R, net P&L
- CSV download
- Embedded transaction cost model reference table

**Insights tab**
- Current market regime: Nifty trend (BULLISH/BEARISH), volatility level, India VIX fear rating
- Strategy-regime fit scores (0–100) showing which strategies suit today's conditions

### 8 · Tip Analyzer
Paste any WhatsApp or Telegram stock tip for an AI-powered credibility check.

1. **Parse** — Claude Haiku extracts ticker, entry, stop-loss, targets, claims, timeframe (regex fallback if API unavailable)
2. **Fetch** — Live price, 52W range, volume, fundamentals via yfinance
3. **Score** — Pump risk detector (flags: "guaranteed returns", "operator backed", "100% sure", etc.), technical alignment, R:R validation
4. **Verdict** — Claude Sonnet writes a plain-English assessment with green/yellow/red flags

Includes 5 pre-loaded sample tips covering common tip archetypes.

## Architecture

```
app.py                     ← Home page + Streamlit multi-page entrypoint
pages/                     ← One file per page (numbered for sidebar order)
signals/
  signal_models.py         ← TradeSignal dataclass
  swing_signals.py         ← 6-strategy swing scanner
  intraday_signals.py      ← 4-strategy intraday scanner
  signal_logger.py         ← SQLite / PostgreSQL persistence + outcome tracking
  outcome_tracker.py       ← Auto-updates outcomes against live prices
  trade_costs.py           ← Brokerage cost calculator (Zerodha-style)
analysis/
  technical.py             ← RSI, MACD, BB, ATR, SMA, EMA, VWAP, Supertrend
  fundamental.py           ← Valuation, profitability, growth, health scores
  sentiment.py             ← Claude Sonnet + VADER sentiment engine
  tip_analyzer.py          ← Two-stage Claude parse + verdict pipeline
data/
  fetcher.py               ← yfinance wrapper with caching
  price_store.py           ← Persistent SQLite OHLCV store (incremental updates)
  news_fetcher.py          ← RSS aggregator (5 sources, 4-hr cache)
  cache_manager.py         ← SQLite TTL cache (survives Streamlit reruns)
  market_status.py         ← NSE trading hours + holiday calendar
config/
  settings.py              ← All thresholds, TTLs, indicator params
  stock_universe.py        ← Nifty 50, Nifty 200, sector → ticker map
ui/
  styles.py                ← Global dark-theme CSS
  charts.py                ← Plotly candlestick, heatmap, gauge components
  components.py            ← Signal card, price ladder, index metric card
notifications/
  telegram.py              ← Signal alerts, morning briefing, market updates
scheduler/
  jobs.py                  ← APScheduler: 6 timed jobs per trading day
```

## Database

**Local (development):** SQLite3 at `data_store/signals.db`, `data_store/prices.db`, `data_store/cache.db`

**Production:** PostgreSQL (Supabase / Neon / self-hosted) via `DATABASE_URL` — the app auto-detects and switches backends.

**signal_log table columns (key ones):**
`signal_id, ticker, strategy, direction, entry_price, stop_loss, target_1, target_2, timeframe, confidence, technical_score, fundamental_score, sentiment_score, outcome, outcome_price, outcome_at, pnl_r, net_pnl_inr, cost_total_inr`

The table and all indexes are created automatically on first run. A unique constraint on `(ticker, strategy, timeframe, signal_date)` prevents duplicate entries.

## Setup

### Prerequisites
- Python 3.11+
- An [Anthropic API key](https://console.anthropic.com) (optional — VADER is the free fallback)
- A Telegram bot token + channel ID (optional — for alerts)
- A PostgreSQL connection string (optional — SQLite is the default)

### Local Development

```bash
git clone https://github.com/haridutt12/indian-stock-screener.git
cd indian-stock-screener
pip install -r requirements.txt

# Create .env (or .streamlit/secrets.toml)
ANTHROPIC_API_KEY=sk-ant-...
TELEGRAM_BOT_TOKEN=123456:ABCdef...
TELEGRAM_CHANNEL_ID=-1001234567890
# DATABASE_URL=postgresql://user:pass@host/db   ← optional

streamlit run app.py
```

### Streamlit Cloud (Production)

1. Fork the repo and connect it to [share.streamlit.io](https://share.streamlit.io)
2. Set secrets in the Streamlit Cloud dashboard:
   ```toml
   ANTHROPIC_API_KEY = "sk-ant-..."
   TELEGRAM_BOT_TOKEN = "123456:ABCdef..."
   TELEGRAM_CHANNEL_ID = "-1001234567890"
   DATABASE_URL = "postgresql://..."
   ```
3. The GitHub Actions workflow (`.github/workflows/wake_app.yml`) keeps the app alive for scheduled jobs

## Key Design Decisions

**One trade per ticker**: If a ticker already has an OPEN position in the signal log, `log_signal()` returns `False` and skips — preventing concurrent duplicate positions in the same stock.

**Golden Cross is event-based**: A cross is only flagged if SMA50 was *below* SMA200 within the last 15 trading days. Stocks that have been in a golden cross state for months are not re-flagged.

**P&L at trigger price**: When a live position hits a target or stop, the P&L card shows the realized value *at that price level*, not the current drifted market price.

**Dual sentiment engine**: Claude provides rich structured analysis (sector-level outlook, stock mentions, catalysts). If the API key is absent, VADER computes a numeric sentiment score from the same headlines — the rest of the app works without any API key.

**Incremental price bootstrapping**: The first run fetches 2 years of daily OHLCV for all tickers (~15 seconds). Every subsequent run reads from SQLite and only fetches the delta since the last update (~30ms for 50 tickers).
